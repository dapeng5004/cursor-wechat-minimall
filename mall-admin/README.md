# 商城后台管理系统

## 项目简介
基于 Vue3 + Element Plus + Vite 开发的商城小程序后台管理系统

## 技术栈
- Vue 3.3.4
- Vue Router 4.2.4
- Pinia 2.1.6
- Element Plus 2.3.8
- Vite 4.4.5
- Axios 1.4.0

## 开发环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

## 安装和运行

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run dev
```

### 生产环境构建
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

## 项目结构
